# LOW_POWER_TECHNIQUES_VHDL
The project addresses power consumption reduction in FPGA circuits by applying low-power techniques at the hardware description level (VHDL), where it is not possible to act on physical parameters such as Vdd scaling or transistor sizing. The optimizations are implemented on a 32-bit adder.  Compared dynamic power dissipation, maximum clock frequency, and hardware resource usage through SAIF file analysis and testbenches, identifying the most energy-efficient and high-performance solution.
