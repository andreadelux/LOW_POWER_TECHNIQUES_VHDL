# LOW_POWER_TECHNIQUES_VHDL
Il progetto ottimizza i consumi energetici in circuiti FPGA tramite tecniche low-power a livello VHDL. Su un sommatore a 32 bit, sono state applicate strategie hardware agendo dove non è possibile modificare parametri fisici. Sono stati confrontati potenza dinamica, frequenza e risorse per individuare la soluzione più efficiente.
